# Cloudflareメモ

フロント側をホスティング

## ドメイン

```
videomng-front-react.pages.dev
```

## デプロイ

1. ホーム画面にアクセスしてアプリ名クリック
2. developmentタブクリック
3. All deploymentsのコミット履歴の右端の三点アイコンからRetry deploymentを選択

## 再ビルド

1. ホーム画面にアクセスしてアプリ名クリック
2. developmentタブクリック
3. 最新のコミット履歴のview detailをクリック
4. manage deploymentのretry deploymentを選択

## 環境変数変更

1. ホーム画面にアクセスしてアプリ名クリック
2. Settingsタブクリック
3. Variables and Secretsから環境変数を設定

## アクセス設定

1. ホーム画面にアクセスして画面左メニューの「Zero Trust」をクリック
2. 画面左メニューの「Access」の「Policies」をクリック
3. 「Allow Specific Users」の三点アイコンから「Configure」を選択
