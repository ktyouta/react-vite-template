# プロジェクト_環境構築手順

## 1. プロジェクトをクローンする

作業フォルダ配下で以下のコマンドを実行：

```bash
git clone https://github.com/GitHubユーザー名/リポジトリ名.git
```

## 2. ライブラリをインストールする

プロジェクトルートに移動して以下のコマンドを実行：

```bash
cd リポジトリ名
npm install
```

## 3. VSCodeの設定を追加

1. VSCodeを開いて、`Ctrl + Shift + P`
2. 「Open Settings (JSON)」を選択し以下を貼り付ける

```json
{
  "files.hotExit": "off",
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "vscode.typescript-language-features"
}
```

## 4. アプリケーションを起動する

```bash
npm run dev
```
